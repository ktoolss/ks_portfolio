KRISTIAN'S PROFILE

Geology to Javascript

Hi. Thanks for taking some time to learn about me. My name is Kristian and I grew up in Houston, TX. I went to the University of Houston for a B.S. of Geology, with two minors of Geophysics and Biology. I was employed as a geoscientist for 5 years, until I used C# to develop a Soils Lab Calculator to automate calculations in the lab I was working in. I discovered how much I really enjoyed coding and decided to enroll in a Bootcamp to enhance my skills.

Software Engineering

I'm currently working as a frontend software engineer at Eagle Eye Networks. EEN focuses on cloud video surveillance. I work in an agile environment with many experienced colleagues. We work together to resolve bugs and develop quality software.

Bootcamp Group Project

We were offered an oppurtunity to develop an application, and pitch this idea to the rest of the class for it to be selected as one of five group projects.

My project was chosen and I became the project leader for a Javascript Blockchain Project. The blockchain is using a client, server, blockchain algorithm source code, and MongoDB/Mongoose for the database. The team is utilizing an agile workflow with two weekly meetings, where we rotate project manager duties.

As the lead developer, I researched the blockchain algorithms and npm modules, as well as connected the database and client. After this, I made a pitch to the class for my project to be accepted as one of the final projects.
